Wall clock vs. exact instant

A wall-clock face with no time zone (PlainDateTime) becomes an exact point in time (ZonedDateTime) once you attach a time zone. Below, a single instant is shown in the local time of several cities at once.

This browser does not support Temporal. Please use Chrome / Edge 144+ or Firefox 139+.

The wall-clock value alone does not tell you which region's 14:30 it is. Only after attaching a time zone does it become a unique instant on Earth.
