Temporal demos

Three demos showing how to use the standard date-time API "Temporal". Sample code for the article on migrating from Date to Temporal.

  • 01. Reservation date-time handling

    Build a confirmed date-time from a reservation date, start time, duration, and store time zone, then convert it to an overseas display time zone.

  • 02. Wall clock vs. exact instant

    Compare a Plain value with no time zone against a ZonedDateTime that maps to a unique instant, shown across several cities.

  • 03. Adding across a DST transition

    See how addition handles a Daylight Saving Time switch where a wall-clock time does not exist.
