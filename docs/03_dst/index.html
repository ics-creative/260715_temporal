Adding across a DST transition

In London, the clocks jump from 01:00 to 02:00 on 29 March 2026, so the 01:00 hour does not exist that day. This demo shows how ZonedDateTime handles addition across that switch.

This browser does not support Temporal. Please use Chrome / Edge 144+ or Firefox 139+.
